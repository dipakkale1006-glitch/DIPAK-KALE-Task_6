# Laundry Mart Login Page

# Project Description
This project is a simple and visually appealing Login Page UI for a Laundry Mart web application. It uses HTML and CSS (Flexbox) to create a clean split-screen layout with an image on one side and a login form on the other.

# Objective
The goal of this project is to:
- Learn Flexbox layout
- Create a centered UI without using margin/padding tricks
- Design a user-friendly login interface

# Features
- Split layout (Left Image + Right Form)
- Fully centered content using Flexbox
- Clean and modern UI design
- Email and Password input fields
- Login button
- Forgot Password link
- Register Now option

# Technologies Used
- HTML5
- CSS3 (Flexbox)

#Project Structure
Laundry-Mart-Login/
├── index.html
└── style.css

# How It Works
The main container uses display: flex to create two sections. The left section contains an image, and the right section contains the login form. Flexbox is used to center content both horizontally and vertically.

# How to Run
1. Download or clone the project
2. Open index.html in your browser

# Output
A login page with a blue image section on the left and a centered login form on the right.

# Learning Outcomes
- Understanding Flexbox properties
- Creating clean layouts
- Designing form UI
- Structuring HTML and CSS
